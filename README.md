### 🏠 House Price Prediction System
**End-to-end regression pipeline for real estate valuation**

A production-ready ML system that predicts residential property prices using advanced feature engineering and ensemble learning techniques. Delivers interpretable results with SHAP value visualizations for stakeholder trust.

**Stack:** `Python` `scikit-learn` `Pandas` `Flask` `Matplotlib`  
**Highlights:** Feature selection · Cross-validation · Model comparison · REST API deployment

[![Repo](https://img.shields.io/badge/View%20Repo-181717?style=flat-square&logo=github)](https://github.com/abrazzaq02)

### 📊 Sales Forecasting & Business Analytics Engine
**Time-series forecasting for inventory and revenue planning**

An automated forecasting solution that uses historical sales data to generate short- and long-term revenue projections, enabling smarter inventory and budgeting decisions for small-to-medium businesses.

**Stack:** `Python` `Prophet` `Pandas` `Matplotlib` `Streamlit`  
**Highlights:** ARIMA/Prophet models · Seasonal decomposition · Interactive dashboard

[![Repo](https://img.shields.io/badge/View%20Repo-181717?style=flat-square&logo=github)](https://github.com/abrazzaq02)

---

### 🤖 Intelligent Recommendation System
**Collaborative + content-based filtering hybrid recommender**

A dual-strategy recommendation engine that combines user behavior patterns with item metadata to deliver personalized suggestions. Designed for e-commerce and content platforms.

**Stack:** `Python` `scikit-learn` `NumPy` `Pandas` `Flask`  
**Highlights:** Cosine similarity · Matrix factorization · Cold-start handling

[![Repo](https://img.shields.io/badge/View%20Repo-181717?style=flat-square&logo=github)](https://github.com/abrazzaq02)

---

### 💬 NLP Text Classifier & Sentiment Engine
**Multi-class text classification and sentiment analysis pipeline**

A complete NLP pipeline for automated text classification, from raw text preprocessing to model deployment. Supports custom categories and multi-label outputs for real-world document tagging.

**Stack:** `Python` `NLTK` `scikit-learn` `TF-IDF` `Flask`  
**Highlights:** Custom tokenizer · Precision/recall reporting · API endpoint · Retraining pipeline
